# Alba&Grazia — Next.js + Firebase + Vercel

Nueva arquitectura moderna para convertir el prototipo en plataforma escalable.

## Incluye
- Next.js App Router
- Firebase preparado
- Login cliente WhatsApp + password usando Firebase Auth
- Firestore preparado para usuarios, productos y pedidos
- Admin demo con Angela Lomas / 0023
- Catálogo premium
- WhatsApp purchase flow
- Diseño luxury italiano

## Setup local
npm install
cp .env.example .env.local
npm run dev

## En Vercel
Añadir variables de entorno desde `.env.example`.
